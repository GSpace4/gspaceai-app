# Common Platform Replacement Patterns

Reference for Gemini — common tools and their Google Workspace alternatives.
The AI must evaluate whether the customer uses advanced features before recommending replacement.

## Scheduling

**Calendly →** Google Calendar appointment schedules + Google Forms intake + Apps Script confirmations + Gmail templates

## Project Management

**Trello / Asana / Monday.com / ClickUp →** Google Sheets project trackers + Google Forms task intake + Apps Script status updates + Google Drive folders + Google Calendar reminders

## Database / No-code

**Airtable →** Google Sheets databases + filtered views + protected ranges + Apps Script automation + Looker Studio dashboards

## Forms / Surveys

**Typeform / Jotform →** Google Forms + Apps Script routing + Google Sheets response tracking + automatic confirmation emails

## Communication

**Slack (lightweight use) →** Google Chat + Gmail groups + Spaces + Calendar workflows

## Knowledge / Docs

**Notion →** Google Sites + Google Docs + Google Drive shared folders + internal knowledge bases

## Email Marketing (lightweight)

**Mailchimp (lightweight) →** Gmail + Google Sheets contact lists + Apps Script mail merge + scheduled follow-ups

## CRM (lightweight)

**HubSpot CRM / Dubsado / HoneyBook (lightweight) →** Google Sheets CRM + Apps Script reminders + Gmail templates + Calendar follow-ups + Looker Studio dashboards

## Important Rules

- Never claim every tool can be replaced.
- Evaluate whether the customer uses advanced features that make the external platform worth keeping.
- Classify each tool as: Keep / Replace / Consolidate / Automate / Investigate.
- Always explain WHY.
