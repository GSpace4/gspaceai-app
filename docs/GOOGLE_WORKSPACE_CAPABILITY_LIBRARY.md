# Google Workspace Capability Library

Systems buildable inside Google Workspace — used by Gemini for replacement analysis.

## Core Tools

| Tool | Use Cases |
|---|---|
| Google Sheets | Databases, trackers, CRMs, dashboards, inventories |
| Google Forms | Lead intake, service requests, surveys, onboarding |
| Google Docs | Proposals, contracts, SOPs, templates |
| Google Drive | File storage, shared folders, document management |
| Gmail | Communication, mail merge, templates, follow-ups |
| Google Calendar | Scheduling, appointment bookings, reminders |
| Google Sites | Internal wikis, knowledge bases, portals |
| Apps Script | Automation, triggers, notifications, data routing |
| Gemini | Drafting, summarizing, classifying, analyzing |
| Looker Studio | Dashboards, reporting, data visualization |
